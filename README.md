Olá!

Esse projeto recria o jogo Snake (1997), lançado nos dispositivos da Nokia.
O jogo utiliza:
  Um arquivo index.html para criação do canvas que é a base principal do projeto. Todos os eventos são baseados dentro do canvas;
  Um arquivo style.css é um arquivo CSS que estiliza somente alguns elementos básicos do canvas;
  Um arquivo cobrinha.js sendo um arquivo de JavaScript que é o cerébro do jogo. Todos os eventos do canvas são manipulados e gerados por aqui.

Para jogar, abra o arquivo index.html e jogue com as setas do teclado.
